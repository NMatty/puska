PRIMARY KEY, FOREIGN KEY

Ez a példa bemutatja, hogyan hozzunk létre egy táblát, valamint használjuk a Primary Key és Foreign Key kulcsokat.

CREATE TABLE Orders
(
    OrderID int NOT NULL,
    OrderNumber int NOT NULL,
    PersonID int,
    PRIMARY KEY (OrderID),
    FOREIGN KEY (PersonID) REFERENCES Persons(PersonID)
);
/*Primary Key: Egyedileg azonosít minden sort a táblában*/ /*Foreign Key: Egy olyan kulcs ami egy 
másik tábla Primary Key-re hivatkozik ezzel összekapcsolja a kettőt, így később egyszerűbb dolgozni 
a táblákkal*/

Lekérdezés szűrése szöveg típusú eredménnyel (Wildcard Characters)

Ez a példa bemutatja, hogyan szűrhetünk szöveges típusú adatokat SQL lekérdezésben a LIKE operátor segítségével. A % jelzés bármilyen karakterek sorozatát jelenti, az _ pedig egyetlen karaktert. Az [] szögletes zárójelekben meghatározott karakterek közül lehet választani, míg a - jel egy tartományt jelöl.

SELECT *
FROM tanarok
WHERE keresztnev LIKE '%a%';
/*Itt minden tanárt megkapunk akinek a keresztneveben van "a" betű*/
SELECT *
FROM orszagok
WHERE fovaros LIKE '_udapest';
/*Itt minden fővárost ki ad, ha az első betűje után udapest-el folytatódik*/
SELECT *
FROM diakok
WHERE nev LIKE '[sfz]%';
/*Azokat a neveket választja ki, amik vagy "s" vagy "f" vagy "z" betűvel kezdődnek*/
SELECT *
FROM autok
WHERE marka LIKE '[a-d]%';
/*a, b, c vagy d betűvel kezdődő eredményeket kapunk csak*/
SQL

Data types

Ez a példa bemutatja az SQL-ben leggyakrabban használt adattípusokat, mint a szöveges típusokat, szám típusokat és dátum típusokat.

/*Az SQL-ben leggyakrabban használt adattípusok:*/ /*Szöveges típusok*/ /* VARCHAR(size) – Változó 
hosszúságú szöveg, pl. VARCHAR(50) – 'Kossuth tér 12.' */ /* TEXT – Nagy mennyiségű szöveg, pl. TEXT
 – 'Ez egy hosszú leírás.' */ /*Szám típusok*/ /* INT(size)/INTEGER(size) – Egész szám, pl. INT – 
100 */ /* DECIMAL(size,d)/DEC(size, d) – Tizedes szám (d = milyen pontosan lehessen megadni), pl. 
DECIMAL(5,2) – 123.45 */ /* FLOAT – Lebegőpontos szám, pl. FLOAT – 3.14 */ /* DOUBLE(p,s) – Nagy 
pontosságú lebegőpontos szám, pl. DOUBLE(10,4) – 12345.6789 */ /* BOOL/BOOLEAN - Logikai változó: ha
 0 az a false, ha nem nulla az true */ /*Dátum típusok*/ /* DATE – Dátum, pl. '2025-04-21' */ /* 
DATETIME – Dátum és idő, pl. '2025-04-21 14:30:00' */ /* TIMESTAMP – Automatikusan kezelt dátum/idő,
 pl. '2025-04-21 14:30:00' */ /* TIME – Időpont, pl. '14:30:00' */ /* YEAR – Év, pl. '2025' */
